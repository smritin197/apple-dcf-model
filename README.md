# Apple Inc. – Three-Statement Financial Model & DCF Valuation

## Project Overview
This project presents a complete three-statement financial model and Discounted Cash Flow (DCF) valuation of Apple Inc. built entirely in Microsoft Excel.

The model integrates the Income Statement, Balance Sheet, and Cash Flow Statement to forecast Apple's financial performance and estimate its intrinsic enterprise value using the DCF valuation methodology.

---

## Project Objectives

- Analyze Apple's historical financial performance
- Build an integrated three-statement financial model
- Forecast financial statements for FY2026–FY2030
- Perform a Discounted Cash Flow (DCF) valuation
- Conduct sensitivity analysis using different WACC and Terminal Growth Rate assumptions

---
